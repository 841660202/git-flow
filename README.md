# git-flow
this is a git-flow
- 在github建立项目仓库，拉取项目
- 项目根目录 npm init -y
- npm i -D husky
- cnpm i -D @commitlint/cli @commitlint/config-conventional
- 配置commitlint.config.js
- 参考 https://blog.csdn.net/wei371522/article/details/84070803
- brew install git
- npx standard-version --first-release
